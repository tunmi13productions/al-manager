Quick Start Example

Get started with AL Manager in just a few lines of code:

from al_manager import Manager

# Create audio manager
manager = Manager()

# Play a simple sound
sound = manager.play("audio.ogg", volume=0.8)

# Play a positioned sound with effects
positioned_sound = manager.play_p("footstep.ogg", x=5, y=0, z=10)
positioned_sound.add_effect('reverb', preset='hall')

# Apply environmental effects
manager.apply_room_acoustics("auditorium")

# Clean up
manager.destroy_all()
